ColorBoxes

Платформер с элементами логической игры. Помоги квадрату найти его брата-близнеца. 
При этом возможно взаимодействовать с окружающим миром с помощью переключения цветов.

Управление
WAD, стрелки - движение
Z или пробел - смена цвета
R - перезапуск уровня

Особенности:
Игра написана под .Net,
используется QuadEngine

Для запуска необходим .Net Framework 4.5

Разработчики:
разработка, дизайн уровней и проч. - Xneg
дизайн уровней - Raven
графический движок - Darthman
