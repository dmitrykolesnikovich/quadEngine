Shattered Void

Игра на конкурс igdc.ru №89 "Скроллер 4"

Герой прорывается сквозь толпы непонятных ему врагов, состоящих из антиматерии, которая незамедлительно аннигилирует
при соприкосновении с героем. Некоторая антиматерия имеет дурацкое свойство притягиваться ко врагу. В темном космосе её
очень сложно увидеть, однако по неведомым причинам антиматерия светится и выдает своими размерами и цветом расположение,
поведение и возможно что-то еще. Наверное настроение =)

Управление:  Клавиатура\геймпад

  Клавиатура:
    Курсорные стрелки - передвижение
    Пробел - стрельба
    Escape - меню

  Геймпад:
    Левый джойстик - передвижение
    Правый курок - стрельба
    Старт - меню

Геймпад рекомендуется XBox совместимый, с вибрацией.
